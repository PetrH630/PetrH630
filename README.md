## 🏆 Main project

### 🏒 HobbyHokej – hobbyhokej.phsoft.cz

Komplexní webová aplikace pro organizaci hobby hokejových zápasů.

### 🔧 Co aplikace řeší:

* registrace hráčů na zápasy (výběr pozice a týmu)
* řízení kapacity a náhradníků
* správa zápasů a výsledků
* notifikační systém (email / SMS / in-app)
* plánované notifikace (scheduler)
* auditní historie změn

### ⚙️ Technické řešení:

* REST API (Spring Boot)
* role-based security (Spring Security)
* databáze + migrace (Flyway)
* asynchronní zpracování (RabbitMQ)
* Docker deployment (VPS)

🔗 GitHub: https://github.com/PetrH630/HobbyHokej-demo
🌐 Demo: https://hokej.phsoft.cz
